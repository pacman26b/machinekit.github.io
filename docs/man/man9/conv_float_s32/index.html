HAL Component — CONV_FLOAT_S32

NAME

conv_float_s32 — Convert a value from float to s32

SYNOPSIS

conv_float_s32

USAGE SYNOPSIS

loadrt conv_float_s32 [count=N|names=name1[,name2…​]]

FUNCTIONS

conv_float_s32.N (requires a floating-point thread) Update 'out' based on 'in'

PINS

in_ float in

out s32 out

out_of_range bit out - TRUE when 'in' is not in the range of s32

PARAMETERS

clamp bit rw - If TRUE, then clamp to the range of s32. If FALSE, then allow the value to "wrap around".

LICENCE

GPL
